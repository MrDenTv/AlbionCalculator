## ✨ Особенности

* 🧮 **Мгновенный расчет:** Считайте профит, учитывая налоги рынка (Премиум / Без премиума).
* 📝 **Умный блокнот (OCR):** Встроенная вкладка для рисования с распознаванием формул и математики. Напишите пример мышкой — калькулятор сам всё посчитает!
* 📚 **История сделок:** Все сделки сохраняются в удобном виде. Следите за своим итоговым профитом и редактируйте цены на лету.
* 🚀 **Моментальный запуск и автообновления:** Программа работает быстро и сама получает свежие версии.

## 📥 Установка

1. Перейдите на вкладку [Releases](../../releases).
2. Скачайте `AlbionCalculator Setup X.X.X.exe`.
3. Запустите установщик, и приложение сразу появится на вашем рабочем столе.

## 🛠 Технологии

* **Electron** — база приложения.
* **Tesseract.js** — нейросеть для распознавания рукописной математики.
* **Vanilla HTML/CSS/JS** — для чистого и быстрого интерфейса без лишних фреймворков.
